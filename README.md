# Part-3.2-software-project
This repository contains my project for part 3.2; a mobile application that helps Solar system designers to choose the right cable size for inverters, fuse boxes and batteries.
Put all these in a folder and name it SolarDesignBuddy
Open the project folder in android studio
Compile it within android studio if you want to make edits 

The app accepts info about the Solar system such as:
PV power in watts
PV voltage in Volts
Isc current
Battery voltage
It then asks for Cable distances in metres such as:
Panels to Fuse Box
Fuse box to Inverter
Inverter to Battery Bank
After which you prompt ot to calculate
The result is both standard sizing and calculated sizing
